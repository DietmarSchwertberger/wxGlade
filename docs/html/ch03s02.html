Tree Window

The tree window show the logical ierarchy of widgets and its child-widgets. For example you can see a panel as a tree's node and the widgets on it as a child nodes.

Figure 3.2. The Tree Window

Usually a frame or a panel contains a sizer, so you often see a sort of panel-sizer-widgets structure. The tree get more complex while you nest sizers on sizers.

You have a visual presentation of your widget tree that you can navigate by mouse, expand and collapse sizers, move or remove widgwts.
