Widget list

Follow the widget list as it appear in the wxGlade main window.

Frame

This prompt for a wxFrame or a wxMDIChildFrame. A vertical wxBoxSizer is appended. In the properties window you can choice the styles and you can add an icon.

Dialog or Panel

This prompt for a wxDialog or a wxPanel in top level. In the properties window you can choice the styles and, for the dialog, you can add an icon.

Panel

This allow to add a panel into a sizer.

In the properties window you can choice the styles.

Splitter window

This produce a wxSplitterWindow and two associated panels as well. You can choice vertical or horizontal splitting.

In the properties window you can choice the styles and the sash position.

Be aware to do not put in a panel under a splitter, a widget too long or big, because in the design window you cannot notice nothing abnormal, but when you run your program, one of two panels will takes all the avalaible space and the other panel will shrink to the minimum size possible.

Notebook

This produce a wxNotebook and one panel for each tab.

In the properties window you can add and remove tabs, that appears in a list.

Don't forget to click on the “Apply” button to transfer changes that you have made in the list to the design window.

Button

This produce a wxButton. You can enter a caption and the “default” flag. You cannot add an image because for this you need a bitmap button (see the section called “Bitmap button”).

Toggle button

This produce a wxToggleButton. You can enter a caption and the status (clicked or not) of the button.

Bitmap button

This produce a wxBitmapButton. You can set the “default” flag on or off. You also can choose the bitmap for the button and, optionally, the bitmap for the disabled status. Refer to the section called “Specifying the path of bitmaps” for bitmap path specifications.

Text control

This produce a wxTextCtrl. In the properties window you can enter the text and also set the style.

Spin control

This produce a wxSpinCtrl. In the properties window you can enter the value, the range and also set the style.

Slider

This produce a wxSlider. In the properties window you can enter the value, the range and also set the style.

Gauge

This produce a wxGauge. In the properties window you can enter the range and set the style.

Static text

This produce a wxStaticText. In the properties window you can enter the text, set the style and tell to wxGlade to store the control as attribute or not.

Check box

This produce a wxCheckBox. In the properties window you can enter the text, and the status, checked or not, of the button.

Radio button

This produce a wxRadioButton. In the properties window you can enter the text, and the status, clicked or not, and the style.

Radio box

This produce a wxRadioBox. In the properties window you can enter the dimension. The style tell if the dimension is the number of rows or columns.

You also can set wich one button is selected with the “Selection” spin starting from 0. You can edit the list of choice, but remember to click on the “Apply” button to consolidate changes.

Choice

This produce a wxChoice. In the properties window you can enter the position of the selected item starting from 0. You can edit the list of choices, but remember to click on the “Apply” button to consolidate changes.

Combo Box

This produce a wxComboBox. In the properties window you can enter the position of the selected item starting from 0. You can edit the list of choices, but remember to click on the “Apply” button to consolidate changes.

List Box

This produce a wxListBox. In the properties window you can enter the position of the selected item starting from 0. You can edit the list of choices, but remember to click on the “Apply” button to consolidate changes.

StaticLine

This produce a wxStaticLine vertical or horizontal. In the properties window you can tell to wxGlade to store or not the object as attribute of the frame class.

Static bitmap

This produce a wxStaticBitmap. You will be prompt for the bitmap path. Refer to the section called “Specifying the path of bitmaps” for bitmap path specifications. In the properties window you can set the style and you can tell to wxGlade to store or not the object as attribute of the frame class.

List Control

This produce a wxListCtrl. In the properties window you can set the style.

Tree Control

This produce a wxListCtrl. In the properties window you can set the style.

Grid

This produce a wxGrid. In the properties window you can set the style, the row number, the label size, the line and background color and the selection mode. You can edit the list of columns, but remember to click on the “Apply” button to consolidate changes. Also you can choose to let wxGlade to create the grid or leave it to the user code.

Custom Widget

When you put a custom widget in the design window you will be prompt for a class name. In the properties window you can set a number of custom attributes that will appear in the costructor call. These attributes have different effect in C++, perl, python or XRC code generation. Four special attributes, $id, $parent, $width and $height returns his value you specify in the “common” tab of the custom widget.

Spacer

When you put a spacer into a sizer slot in the design window you will be prompt for the size; wxGlade will generate the code to set a empty space in that slot of the sizer.
