Requirements

The wxGlade requirements are:

  • Python 2 - at least 2.3 or any later version of Python 2

  • wxPython 2.6 or any later version

  • wxWidgets 2.6 or any later version, the wxWidgets are often bundled with wxPython

wxWidgets are available at http://www.wxwidgets.org and wxPython at http://www.wxpython.org.
