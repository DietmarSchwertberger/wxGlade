Properties Window

The properties window let you see and edit the properties that applies to the element that you select. This window consist in three tabs. The first tab contains the common properties that apply to all widgets.

Figure 3.6. The Properties Window

As shown in Figure 3.6 The common properties are related to name, size, colors and fonts.

The second tab is related to properties that controls position and resizing inside the sizer.

Figure 3.7. The Properties Window

You can check or uncheck any option related to the placement in the sizer. Many widgets may have a default value of 3 in the "Border" property in the preferences dialog, see the section called “Preferences Dialog”. If you let widget to have a default border, also the wxALL option is checked.
