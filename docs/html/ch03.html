Main Window

The main window hosts the menu and the widgets choice buttons.

Figure 3.1. The Main Window

If you pass the mouse pointer over a button a tooltip show the button's description. The ``frame'' button and the ``Dialog'' button show up a dialog to add a frame, a dialog or a panel to your project. Other buttons in the main window adds widgets in a form. When you click on such button, the cursor pointer change in an arrow, then you can click on a sizer's empty cell and add the widget to it.
